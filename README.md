## Getting started

#### Prerequisites

- Ruby
- Ruby on Rails
- PostgreSQL

#### Setup

Clone [this](t) repo

- Navigate to the directory and run `bundle install` to install the dependencies.
- Run `rails db:create` to create the database.
- Run `rails db:migrate` to migrate the database.
- Run `rails s` to start the server.
